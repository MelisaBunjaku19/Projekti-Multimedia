Game Engines

Unity, Unreal Engine, Godot

Game engines provide the framework for building games. Unity is known for its versatility and is widely used for both 2D and 3D game development. Unreal Engine offers high-fidelity graphics and is popular in AAA game development. Godot is open-source and highly customizable, making it a great choice for indie developers.

Programming Languages

C#, C++, JavaScript

C# is primarily used with Unity for its simplicity and efficiency. C++ is used with Unreal Engine due to its high performance and control over system resources. JavaScript is often used for web-based games and game development frameworks like Phaser.

Graphics Tools

Blender, Maya, Photoshop

Blender is a free and open-source tool for creating 3D models and animations. Maya is a professional-grade tool used in the film and game industry for complex 3D modeling and animation. Photoshop is widely used for creating textures, concept art, and other 2D assets.
